# DSA Visualizer 🚀

An interactive web-based platform designed to demystify complex Data Structures and Algorithms through real-time visualization. This tool bridges the gap between theoretical code and visual intuition, allowing users to watch how data moves and transforms.

---

## 🛠 Features

* **Interactive Sorting:** Visualize algorithms like **Bubble Sort**, **Quick Sort**, and **Merge Sort** with adjustable speeds and array sizes.
* **Dynamic Data Structures:** Manually add, remove, or search for elements in **Linked Lists**, **Stacks**, and **Queues**.
* **Pathfinding Visualization:** Watch how **Breadth-First Search (BFS)** and **Depth-First Search (DFS)** navigate through graph structures.
* **Step-by-Step Execution:** Pause and play animations to understand the logic at every iteration.

## 💻 Tech Stack

* **Frontend:** HTML5, CSS3, JavaScript (ES6+)
* **Logic:** Custom DSA implementations in JavaScript to handle state changes and animations.
* **Deployment:** GitHub Pages

## 🚀 Getting Started

### Prerequisites
* A modern web browser (Chrome, Firefox, or Edge).
* Live Server extension (if running locally via VS Code).

### Installation
1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/PiRATEO1/DSAVisual.github.io.git](https://github.com/PiRATEO1/DSAVisual.github.io.git)
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd DSAVisual.github.io
    ```
3.  **Launch the application:**
    Open `index.html` in your browser.
